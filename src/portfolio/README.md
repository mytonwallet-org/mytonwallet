# MyTonWallet Portfolio

This is a standalone dapp for MyTonWallet's professional features.

## Development

To run the Portfolio dapp in development mode:

```bash
npm run portfolio:dev
```

This will start the development server on port 4324.

## Building

To build the Portfolio dapp for production:

```bash
npm run portfolio:build
```

For development build:

```bash
npm run portfolio:build:dev
```

## Structure

The Portfolio dapp follows a similar structure to the main MyTonWallet project:

- `/components`: UI components specific to the Portfolio dapp
- `/utils`: Utility functions and helpers
- `/public`: Static assets and public files

## Configuration

The Portfolio dapp configuration is located in `config.ts`.
